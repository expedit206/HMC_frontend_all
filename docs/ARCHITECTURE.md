# Frontend Architecture

## Vue.js Project Structure

Structure du projet frontend alignée avec le design system Home Cameroon.

### Dossiers principaux

- **src/pages/**: Pages principales de l'application
  - admin/: Dashboard administrateur et gestion
  - agent/: Tableau de bord agent immobilier
  - auth/: Authentification (connexion, inscription, récupération mot de passe)
  - bailleur/: Dashboard bailleur et gestion de biens
  - client/: Dashboard client
  - locataire/: Dashboard locataire et gestion de locations
  - prestataire/: Dashboard prestataire et services
  - public/: Pages publiques

- **src/components/**: Composants réutilisables
  - Header, Footer, Layout components
  - UI components (buttons, cards, forms)

- **src/layouts/**: Layouts de base
  - DefaultLayout: Layout par défaut avec header/footer

- **src/market-place/**: Marketplace components

- **src/documents/templates/**: Modèles de documents

- **public/**: Fichiers statiques
  - css/: Feuilles de style
  - images/: Images par catégorie
